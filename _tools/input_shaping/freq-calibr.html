Input Shaping Calibration Pattern

Caution!
Works for cartesian printers with rectangular bed. Core-XY is not tested.

Use this form to generate G-code that you can use to calibrate your Input Shaping Damping Frequency and Zeta/Damping factor. Default values apply to standard PLA with a 0.4mm nozzle.

The generated G-code files print frequency sweeps (zigzag patterns with increasing speeds) to find resonant frequencies (frequencies to dampen) and zeta/damping factors for X-axis and Y-axis.

3 separate files are generated for download:

    Damping Frequency to find resonant frequency for X-axis and Y-axis (shown on right side)
    Zeta/damping factor for X-axis (not shown)
    Zeta/damping factor for Y-axis (not shown)

Press the Generate G-code button followed by each Download .. as file to save the results.

warning

Notes on the settings:

  • Max X Frequency and Max Y Frequency should be high enough to reach resonant frequency. A printer with horizontally moving bed (aka. bedslinger) usually has lower resonant frequency on Y-axis due to higher moving mass. Alter values after first test print as needed to match the printer.
  • Use Bed Leveling requires a probe.
  • Printer and other parts (e.g. filament spool) may resonate at several frequencies.
  • Printer may skip layers at or above resonant frequency. It is normal behaviour.
  • 60Hz frequency equals 120 mm/s print speed. With default 0.5 mm line width and 0.2 mm layer height, that equals to 12 mm^3/s extrusion flow. Nozzle Temperature can be raised to increase extrusion flow.

Damping Frequency Pattern

Zeta/Damping Factor Pattern

  • Select most uniform pattern from the print. Damping factor increases by 0.05 for each pattern.
